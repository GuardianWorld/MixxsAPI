#			LootAPI Config, be careful touching here.

#		ItemList = Necessary for every item that will be added to be placed here, it will be used to call the rest.
#		idItem"ItemListName": Will select the ItemID for said item.
#		Texture"ItemListName": Will show the path for said item texture. (Its minecraft canonicalPath + /config) Per default the texture used is the PlaceHolder.
#		Type"ItemListName": Will determine the type to be added. Currently we have:
#		SetMaxStack"ItemListName": Will set the max stack size for said item. Only works for certain cases.
#		SetName"ItemListName": To place the FULL name of the object.
#		- Item (Resource)
#		To do: - Pickaxe, Spade, Sword, Hoe, Food, Bucket, Door.

#		LootAPI and receiving the ItemTable).
